# trace ssl

Captures data on read/recv or write/send functions of OpenSSL, GnuTLS, NSS and Libcrypto

## Getting started

Running the gadget:

<Tabs groupId="env">
    <TabItem value="kubectl-gadget" label="kubectl gadget">
        ```bash
        $ kubectl gadget run ghcr.io/inspektor-gadget/gadget/trace_ssl:latest [flags]
        ```
        :::tip
        If you want to try the gadget without installation you can check [kubectl node debug](https://inspektor-gadget.io/docs/latest/quick-start#kubectl-node-debug).
        :::
    </TabItem>

    <TabItem value="ig" label="ig">
        ```bash
        $ sudo ig run ghcr.io/inspektor-gadget/gadget/trace_ssl:latest [flags]
        ```
        :::tip
        If you want to try the gadget without installation you can check [run in a container](https://inspektor-gadget.io/docs/latest/quick-start#run-in-a-container).
        :::
    </TabItem>
</Tabs>

## Flags

### `--record-data`

controls whether the gadget will send data to userspace

Default value: "true"
