# trace signal

trace signal

## Getting started

Running the gadget:

<Tabs groupId="env">
    <TabItem value="kubectl-gadget" label="kubectl gadget">
        ```bash
        $ kubectl gadget run ghcr.io/inspektor-gadget/gadget/trace_signal:latest [flags]
        ```
        :::tip
        If you want to try the gadget without installation you can check [kubectl node debug](https://inspektor-gadget.io/docs/latest/quick-start#kubectl-node-debug).
        :::
    </TabItem>

    <TabItem value="ig" label="ig">
        ```bash
        $ sudo ig run ghcr.io/inspektor-gadget/gadget/trace_signal:latest [flags]
        ```
        :::tip
        If you want to try the gadget without installation you can check [run in a container](https://inspektor-gadget.io/docs/latest/quick-start#run-in-a-container).
        :::
    </TabItem>
</Tabs>

## Flags

### `--failed`

Show only failed events

Default value: "false"

### `--pid`

Show only events generated by processes with this pid

Default value: ""

### `--kill-only`

Show only events generated by kill family syscalls

Default value: "false"

### `--signal`

Show only events generated by this signal

Default value: ""
